This is a sandbox for playing with [red-black trees](https://www.wikiwand.com/en/articles/Red%E2%80%93black_tree).

<img width="847" height="554" alt="Captura de pantalla 2025-11-10 a la(s) 18 35 19" src="https://github.com/user-attachments/assets/82f5ddfa-7b82-4aee-80ed-a4b2ab8afebb" />

So far you can:
 - add and remove edges
 - move nodes around
 - toggle red/black status.

It's not finished or deployed. If you want to try it out, you have to clone the repo and build with `bun dev`.
